# Icon Set Generator (Venngage)

This app generates a set of 4 consistent 512x512 icons using the Replicate
**black-forest-labs/flux-schnell** model. You can:

- Enter a prompt (e.g. "Toys")
- Choose from 5 preset styles
- Optionally provide brand HEX colors
- Generate 4 themed icons in a consistent style
- Download individual icons or all as a ZIP

## Tech Stack

- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express
- Image Gen: Replicate (black-forest-labs/flux-schnell)
